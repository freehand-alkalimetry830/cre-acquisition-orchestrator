# Example Output

This directory contains the complete output from a single simulation run of the Parkview Apartments deal (parkview-2026-001, Core-Plus scenario).

## Contents

- **phase-outputs/** — JSON output from each of the 5 pipeline phases
- **reports/** — Generated analysis reports and agent workpapers
  - `final-report.md` — The final acquisition recommendation
  - `{phase}/` — Individual agent workpapers showing each specialist's analysis
- **master-checkpoint.json** — The pipeline's master state checkpoint
- **story-events.ndjson** — Structured event log of the entire pipeline execution
- **documents-manifest.json** — Registry of all documents produced during the run

## How This Was Generated

This output was generated by running:

```bash
npm run demo
```

The simulation uses deterministic seeded random number generation, so the same scenario + seed will always produce identical results.
