## 🧭 What this is

cre-acquisition-orchestrator is a Windows app for commercial real estate acquisitions. It helps you track deals, review due diligence, and follow each phase of the process in one place.

It uses 31 agents, 5 deal phases, and 8 domain skills to organize work. You do not need API keys to run it.

## 🏗️ Deal phases

The app is set up around 5 common acquisition phases:

1. Deal intake
2. Early review
3. Due diligence
4. Underwriting
5. Final decision

Each phase helps you keep the next task clear. This cuts down on missed steps and scattered notes.

## 📊 Dashboard view

The real-time dashboard shows deal status, task flow, and agent output in one place.

You can use it to:

- See what stage a deal is in
- Check what work is done
- Spot blocked tasks
- Review notes from different agents
- Keep the team aligned on the same deal

## 🔧 How the app works

The app uses multiple agents that each handle a part of the acquisition flow. One agent may review a document, while another checks a task, and another helps with underwriting.

This setup helps keep work split into small parts. That makes the process easier to follow on a Windows desktop.
